
About The Project

concordance system, made as part of university course. this is a JS UI and SQL cloud based DB.
 no need of any presetup or installation, the UI is avilibale at the address:
 https://pensive-spence-70b231.netlify.app

Features

* on-line UI, accessible from any computer
* Insert new e-book as a .txt file from Gutenberg Project
* Filter the books by author, title, or content
* Filter word appearances using many parameters
* Define word groups for cherry-picked filtering
* Define and search phrases in text
* General statistics about the words and books
